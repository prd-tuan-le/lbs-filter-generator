How it works?

  1. Enter multiple lines of task's ids here and it will be converted to filter values on the next box.
  2. Change the quantity box to a number represents how many tasks you want to be displayed for each filter.
  3. Press the "GO!" button to generate filter values on the second box.
  4. Copy the result on the second box, each generated row is a value for the filter.
    DO NOT COPY ALL OF THE GENERATED VALUES!
  5. Paste it into your (or reviewer's) project' filter.
    Create filter as the following screenshot:

    Note: id_1|id_3|id_3 is one row you copy on the next box.
  6. Cycle through the filtered tasks and do what you want
